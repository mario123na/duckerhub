Instalacion de PHPMyadmin

Instalaremos PHPMyadmin para una mejor gestion de BD que hemos instalado con mysql, con el comando:

apt-get install phpmyadmin

Una vez instalado el phpmyadmin para probarlo debemos hacer lo mismo que al probar el servidor apache, nos vamos a http://localhost o http//127.0.0.1 o la ip de nuestro servidor, pero esta vez colocaremos /phpmyadmin, en el navegador se ingresado asi: http://127.0.0.1/phpmyadmin y nos mostrara la siguiente pantalla:
