Configuracion Mysql

Si deseamos habilitar los logs de mysql se debe hacer lo siguiente:

Vamos y abrimos el archivo:

/etc/mysql/my.cnf

Y dentro del descomentaremos las siguientes líneas:

#general_log_file        = /var/log/mysql/mysql.log
#general_log             = 1
#log_bin                 = /var/log/mysql/mysql-bin.log

Y al final de este mismo archivo my.cnf agregaremos las siguientes líneas

long_query_time = 10
server-id = 0
slow_query_log_file = "/var/log/mysql/mysql-slow.log"
slow-query-log

Luego guardamos los cambios pero antes de reiniciar el servicio debemos crear estos archivos que se guardaran los logs

Nos vamos a la dirección /var/log/mysql

con el comando

cd /var/log/mysql

Dentro de esta carpeta crearemos tres archivos, con los comandos:

touch mysql.log mysql-bin.log mysql-slow.log

Una vez creados debemos asignarle como dueño de los archivos el usuario mysql que se crea la instalar la BD, con el comando:

chown mysql:mysql mysql.log mysql-slow.log mysql-bin.log

Y ahora con todas estas configuraciones podemos reiniciar el servicio de mysql, con el comando:

/etc/init.d/mysql reload
/etc/init.d/mysql restart
